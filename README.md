# 🦅 Harpia Studios

## 📌 Sobre a Empresa

A **Harpia Studios** é uma empresa focada no desenvolvimento de soluções educacionais e interativas com propósito social, unindo tecnologia, acessibilidade e inovação.

Nosso objetivo é criar experiências digitais que respeitem diferentes formas de aprendizado, promovendo inclusão e desenvolvimento cognitivo por meio de jogos e ferramentas inteligentes.

---

## 🎯 Missão

Desenvolver produtos digitais acessíveis que utilizem tecnologia como ferramenta de inclusão, aprendizado e desenvolvimento humano.

## 🌍 Visão

Ser referência na criação de softwares educacionais inclusivos, especialmente voltados para crianças neurodivergentes.

## 💡 Valores

* Inclusão em primeiro lugar
* Design centrado no usuário
* Acessibilidade como padrão, não diferencial
* Inovação com propósito
* Responsabilidade social

# 🚀 Equipe Spark

## 🤖 Projeto: Spark — Narrative Coding Game (Foco TEA)

O **Spark** é um software de alfabetização lógica voltado para crianças de **8 a 12 anos dentro do Espectro Autista (TEA)**.

### 🧩 Proposta do Produto

O projeto utiliza conceitos de programação visual para transformar o aprendizado em uma experiência previsível, segura e estimulante.

### 🔍 Diferenciais

* **Comunicação Visual Dirigida**
  Interface baseada em ícones e narrativa visual, reduzindo dependência de texto.

* **Conforto Sensorial**

  * Paleta de cores suave
  * Ausência de estímulos agressivos
  * Controle de áudio pelo usuário

* **Rotina e Previsibilidade**
  Estrutura consistente de fases, promovendo segurança cognitiva.

* **Foco no Raciocínio Lógico**
  Sistema de blocos (drag-and-click) elimina barreiras de sintaxe.
